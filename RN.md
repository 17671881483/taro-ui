# Taro UI For RN


### 开发


#### 壳

下载官方壳应用(NervJS/taro-native-shell)[https://github.com/NervJS/taro-native-shell] 选择与 taro-ui react-native 版本一致的分支


```
# 保证版本一样，lerna 对 packages 进行 symlink
lerna bootstrap
```

#### 组件开发
```
cd packages/taro-ui/rn
```

#### 发布
rn 源码发布

#### DEMO 启动
```
cd packages/taro-ui-demo-rn
yarn dev:rn
```
